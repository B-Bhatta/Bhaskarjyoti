BBCare - Healthcare Booking System

BBCare is an online platform for booking healthcare services like doctor appointments, lab tests, and blood collection, with automated email confirmations. It includes:

  • Responsive HTML forms for user input.
  • Backend server to process bookings and send emails.
  • Real-time email notifications using Gmail SMTP.
  • Deployed on Render for live access.

Technologies Used: HTML, CSS, JavaScript, Node.js, Express.js, Nodemailer, Render.
